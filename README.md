# Kaggle Store Sales: 統計的機械学習プロジェクト - Step 2
Step 2では、エクアドルの小売チェーンの売上データを用い、推論統計学の手法を用いてプロモーション戦略の効果と傾向を定量的に検証しました。

## 実施事項と分析結果

### 1. 確率分布の確認 (正規性の検定)
- **手法**: ヒストグラムおよびQ-Qプロットによる可視化。
- **結果**: 売上データは高価格帯に向かって長い裾を持つ分布（Heavy Tail）を示し、正規分布から大きく逸脱していることを確認しました。この結果は、後のモデリングにおいて対数変換などのデータ前処理が必要であることを示唆しています。

### 2. 平均売上の推定 (信頼区間)
- **結果**: 2017年の全店舗における平均売上の95%信頼区間は **[475.95, 484.30]** と算出されました。標本サイズが非常に大きいため、平均値の推定精度が極めて高いことが分かります。

### 3. プロモーション効果の検証 (ウェルチのt検定)
- **問い**: プロモーションの有無で売上の平均値に差はあるか？
- **統計量**: t統計量 = 213.82, p値 = 0.0000e+00
- **結論**: p値が有意水準0.05を大幅に下回ったため、帰無仮説を棄却。プロモーションは売上に対して統計的に**圧倒的に有意な正の影響**を与えていることが証明されました。

### 4. 店舗タイプと戦略の関連性 (カイ二乗検定)
- **問い**: 店舗タイプ(A〜E)によってプロモーションの実施頻度に偏りはあるか？
- **分割表の実績**:
    | type | Promo OFF (0) | Promo ON (1) |
    | :--- | :--- | :--- |
    | A | 34,425 | 32,994 |
    | B | 31,821 | 28,107 |
    | C | 66,026 | 46,339 |
    | D | 72,692 | 62,146 |
    | E | 16,483 | 13,481 |
- **統計量**: カイ二乗統計量 = 1204.79, p値 = 1.4578e-259
- **結論**: 店舗タイプとプロモーション実施には極めて強い関連性があります。特定の店舗タイプ（例えばタイプDやAなど）において、より重点的にプロモーション施策が打たれている傾向がデータから裏付けられました。
